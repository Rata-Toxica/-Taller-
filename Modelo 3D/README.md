
# Diseño 3D del Encapsulado

Esta carpeta contiene los archivos del encapsulado diseñado para el sistema inteligente de monitoreo de humedad y recomendación de riego.

## Archivos de la caja

- `CajaTaller.f3d`: archivo editable de Autodesk Fusion 360.
- `CajaTaller.step`: modelo CAD compatible con otros programas.
- `CajaTaller.stl`: archivo preparado para impresión 3D.

## Archivos de la tapa

- `TapaTaller.f3d`: archivo editable de Autodesk Fusion 360.
- `TapaTaller.step`: modelo CAD compatible con otros programas.
- `TapaTaller.stl`: archivo preparado para impresión 3D.

## Objetivo del diseño

El encapsulado fue diseñado para proteger y organizar los componentes electrónicos del sistema de riego inteligente.

En su interior se consideran espacios para:

- ESP32.
- Módulo relé.
- Fuente de alimentación.
- Adaptador de conexiones.
- Cableado del sensor de humedad.
- Cableado del sistema de riego.

El diseño incluye una caja principal, una tapa desmontable y aberturas para las conexiones externas.

## Software utilizado

El modelo fue desarrollado utilizando Autodesk Fusion 360.

## Formatos disponibles

- `.f3d`: archivo editable original de Fusion 360.
- `.step`: archivo CAD compatible con distintos programas.
- `.stl`: archivo utilizado para impresión 3D.

## Unidades

Los modelos fueron diseñados utilizando milímetros.

## Recomendaciones de impresión

- Material recomendado: PETG o PLA.
- Altura de capa sugerida: 0,20 mm.
- Relleno sugerido: entre 15% y 25%.
- Revisar la orientación de la caja y la tapa antes de imprimir.
