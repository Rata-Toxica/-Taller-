# Documentación

Esta carpeta contiene la documentación del proyecto **Sistema Inteligente de Monitoreo de Humedad y Recomendación de Riego**.

## Contenido

- Presentación final del proyecto.
- Manual de usuario.
- Documentación complementaria.

## Descripción

En esta carpeta se almacena la documentación desarrollada durante el proyecto, incluyendo la presentación utilizada para la exposición y el manual de usuario, con el objetivo de facilitar la comprensión del funcionamiento del sistema y su utilización.
