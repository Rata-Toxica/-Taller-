Integrantes:

El objetivo de este proyecto es poder crear un dispositivo que ayude a la produccion agricultora mediante una herramienta de detección de humedad, asegurando sobre todo un riego consistente y sin llegar al exceso.

Para funcionar este programa: 
Instale las funcionalidades de Arduino 1.8
Establezca la conexión a una señal al compilar app.py
Identifique la IP que esta usada
modifique el codigo para que coincida con la IP
